# Startup Production Profile

Use this profile for production apps with active users, CI/CD, analytics, and business-critical releases.

## Extra Focus

- Environment separation
- CI secrets
- Rollback readiness
- Crash-free release criteria
- Analytics and consent consistency
- Store metadata and privacy label alignment

## Default Risk Posture

Treat release blockers seriously. Prefer explicit owners, validation steps, and staged rollout recommendations.
